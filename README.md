fetch와 pull의 차이 
fetch : 원격 저장소의 데이터를 로컬에 가져오기만 하기 
pull: 원격 저장소의 내용을 가져와 자동으로 병합 작업을 실행 
즉, 단순히 원격 저장소의 내용을 확인만 하고 로컬 데이터와 병합은 하고 싶지 않은 경우
